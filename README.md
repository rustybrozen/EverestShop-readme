   ```bash
   git clone https://github.com/rustybrozen/EverestShop && cd EverestShop
   ```

2. **Create environment file:**


   ```bash
   sudo docker compose down -v
   sudo docker compose up --build -d
   ```
